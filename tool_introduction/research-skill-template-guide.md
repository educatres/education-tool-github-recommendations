# 從範本建立研究用 AI Skill

## 簡短描述

以研究工作流程為情境，說明如何從 Skill 範本建立可重複使用的 AI Agent 工作手冊。

## 教育工作者摘要

這是一份面向研究人員與研究方法教學的 AI Skill 建立指南，將投影片內容整理成可在 GitHub 閱讀與維護的教學文件。它以 skill-template.zip 為起點，說明如何用自然語言描述研究任務，並請 AI 產生 SKILL.md、agents、scripts、references 與 assets 等資料夾內容。教師可用它帶領研究生或研究團隊理解 AI Agent 的任務拆解、品質檢查、參考資料管理與輸出模板設計，也適合作為建立文獻回顧、IRB 文件整理、資料清理或研究報告輔助流程的入門教材。

## 教學用途

- 研究方法、AI 素養或數位研究工具課程中示範如何把研究 SOP 轉成 AI Skill
- 協助研究生將文獻搜尋、資料整理、報告撰寫與品質檢查拆成清楚的 Agent 分工
- 作為研究室建立可重複使用 AI 工作流程、文件模板與自動化腳本的起點

## 導入注意

- Skill 產出的研究建議、文獻摘要與文件草稿仍需由研究者查證與審閱
- 若流程涉及人體研究、個資、未公開資料或敏感研究資料，需先確認 IRB、資料保護與校內規範
- 初次導入時宜先選擇低風險、可人工驗證的小型任務測試，再逐步擴充到正式研究流程

## 啟動或安裝方式

閱讀 GitHub README 並下載 skill-template.zip 後，先用自然語言寫下想建立的研究任務、資料來源、輸出格式與品質檢查規則，再請具備檔案產生能力的 AI 依範本建立 Skill。完成後應在測試案例中檢查觸發條件、Agent 分工、參考資料讀取與輸出格式是否符合研究情境。
