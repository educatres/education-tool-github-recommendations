# 三角函數實驗室

## 簡短描述

用單位圓、Magic Hexagon 與六個週期波形探索廣義角、同界角與六種三角函數。

## 教育工作者摘要

三角函數實驗室是 Classroom Lab tzk 的數學互動頁，學生可以拖曳單位圓上的點或用鍵盤改變廣義角，並在度度量與徑度量之間切換。工具同步顯示同界角公式、座標投影、sin、cos、tan、cot、sec、csc 的函數值、Magic Hexagon 關係與六張週期圖。它適合把單位圓定義、特殊角、倒數關係、平方關係與週期性整合在同一個可操作畫面中。

## 教學用途

- 在三角函數單元中說明單位圓、廣義角、同界角與徑度量
- 讓學生拖曳角度並觀察六個三角函數值與週期圖同步變化
- 搭配 Magic Hexagon 討論倒數關係與 sin^2θ + cos^2θ = 1 等恆等式

## 導入注意

- 六種三角函數同時呈現資訊量較大，建議分階段引導學生觀察
- 分母為 0 時會出現未定義，需補充為何 tan、sec、cot、csc 在特定角度不存在
- 根號精確值只在特殊角才適合使用，鄰近角度應以近似值理解

## 啟動或安裝方式

可直接開啟 GitHub Pages 的三角函數實驗室頁面。若要本機使用，可下載 classroom-lab-tzk 專案後以靜態伺服器開啟，或從工具總頁進入 trigfun/trigfun.html。
