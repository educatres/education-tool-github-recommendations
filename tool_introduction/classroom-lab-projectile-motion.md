# 拋體運動實驗室

## 簡短描述

以互動式方格紙示範水平地表拋體運動，呈現初速分量、飛行時間、水平位移與最高點高度。

## 教育工作者摘要

拋體運動實驗室是 Classroom Lab tzk 中的物理模擬頁，可調整初速、發射角與發射點高度，並以固定水平地表 y = 0 展示拋物線軌跡。工具會把 V0 分解為 Vx 與 Vy，並在解算結果區逐卡呈現水平初速度、垂直初速度、飛行時間、最高點時間、水平位移與最高點垂直位移。它適合讓學生連結公式、向量分解與動畫軌跡，理解水平等速與垂直等加速度如何共同形成拋體運動。

## 教學用途

- 在國中自然科或高中物理入門中示範拋體運動的水平與垂直分解
- 讓學生改變初速、角度或發射高度，預測飛行時間、射程與最高點變化
- 搭配公式卡討論 v = v0 + at、x = v0t + 1/2at^2 與 v^2 = v0^2 + 2ax 的代入方式

## 導入注意

- 模型採水平地表與理想化重力，不包含空氣阻力、風或真實投擲姿態
- 建議學生每次只改變一個參數，否則難以判讀初速、角度與高度的個別影響
- 公式推導仍需教師引導，避免學生只記結果卡而未理解水平與垂直方向的差異

## 啟動或安裝方式

可直接開啟 GitHub Pages 的拋體運動實驗室頁面。若要本機使用，可下載 classroom-lab-tzk 專案後以靜態伺服器開啟，或從工具總頁進入 projectilemotion/projectilemotion.html。
