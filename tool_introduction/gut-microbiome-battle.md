# 菌叢爭霸戰：腸道好菌壞菌互動教材

## 簡短描述

國中生物腸道菌叢互動教材，結合菌種圖鑑、七日生態挑戰與概念測驗，引導學生理解菌株差異、生態平衡及抗生素的正確使用。

## 教育工作者摘要

菌叢爭霸戰是一套可直接部署到 GitHub Pages 的純前端生物教材。學生可透過 6 種代表性腸道菌的顯微影像與資訊卡，辨識有益或共生菌、條件致病菌與外來病原，理解同一物種不同菌株可能具有不同作用。七日菌叢生態挑戰讓學生每天組合飲食、抗生素、優酪乳或益生菌、作息與運動選擇，觀察菌叢多樣性、腸道屏障、短鏈脂肪酸、壞菌壓力、發炎指數與恢復力的變化。教材另有概念快問快答、中文延伸影片、疾病新聞與資料來源頁，適合用於國中生物、健康教育或高中生命科學的腸道微生物與生態平衡單元。

## 教學用途

- 在國中生物、健康教育或高中生命科學課程中介紹腸道菌叢、共生關係、條件致病與菌株差異
- 讓學生比較高纖飲食、發酵食品、抗生素使用、益生菌、睡眠與運動等選擇對菌叢生態指標的影響
- 搭配七日挑戰、資訊卡與概念測驗，討論為何維持多樣性與生態平衡比單純消滅所有「壞菌」更重要

## 導入注意

- 七日挑戰是簡化的教學模型，遊戲數值與結果不代表臨床建議，也不能用來診斷或治療疾病
- 「好菌／壞菌」僅是入門語彙，教師應提醒學生菌種作用會受到菌株、數量、位置、宿主與環境條件影響
- 抗生素是重要的治療工具，教材重點是遵照醫療專業指示正確使用，不應讓學生解讀成抗生素一律有害
- 顯微影像、影片與延伸資料來自不同外部來源，公開展示或改作前應依資料來源頁逐項確認署名與授權要求

## 啟動或安裝方式

可直接開啟 GitHub Pages 線上版本使用。若要本機測試，請下載或 clone 專案後，在專案資料夾執行 python3 -m http.server 8000，再開啟 http://localhost:8000；由於網頁會讀取 data/bacteria.json，不建議直接以 file:// 開啟 index.html。
